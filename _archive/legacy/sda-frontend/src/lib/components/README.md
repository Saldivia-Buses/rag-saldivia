# Components

Reusable UI components for the SDA Frontend, organized by category.

## Subfolders

- **`chat/`** — Chat-specific components: message rendering, chat input, crossdoc progress indicators, decomposition view, history panel, sources panel, markdown renderer.

- **`ui/`** — Generic reusable primitives: Button, Badge, Modal, Toast, Skeleton, Card, Input. These are design-system-level components used throughout the app.

- **`sidebar/`** — Navigation sidebar components: Sidebar container and SidebarItem for navigation links.

- **`layout/`** — Layout-level structural components: app shell sidebar that wraps page content and provides the navigation structure.

Each subfolder has its own README with detailed information about the components it contains.
