Preguntas relacionadas

Multi-colección en query
Selector de múltiples colecciones antes de enviar
●●
Modos de foco

●
Voz en input
Web Speech API, transcripción en tiempo real

●●●
Etiquetas en sesiones + bulk

●●
Templates de query

●●
Respuestas guardadas / pinned

Compartir sesión

●●
Analytics dashboard
Colecciones más consultadas, feedback scores, queries por hora
