Colaboración · Automatización · Power user · Inteligencia del sistema

11

Vista dividida — de VS Code / Cursor

Dos chats lado a lado en la misma pantalla. Útil para comparar respuestas de distintas colecciones, o tener un contexto de referencia mientras trabajás en otro. Ideal para analistas.

12

Búsqueda universal — de Spotlight / Raycast

Una sola búsqueda indexa todo: sesiones, documentos en colecciones, respuestas guardadas, usuarios (admin), templates. Resultados en tiempo real con preview. Va dentro del Command Palette o como standalone Cmd+F.

13

Analytics de uso — de Amplitude / Mixpanel

Dashboard para admins: qué colecciones se consultan más, qué usuarios son más activos, distribución de feedback positivo/negativo, queries por hora, temas recurrentes. Datos para mejorar el sistema.

14

Auto-ingesta desde fuentes externas — de Zapier / n8n

Conectar Google Drive, SharePoint o Confluence a una colección. Los documentos nuevos se ingestan automáticamente. El admin configura la fuente y el schedule (tiempo real, cada hora, diario).

15

Informes automáticos programados — de Digest / Substack

El usuario configura un query recurrente: "Todos los lunes a las 9am, resumí novedades de la colección Legal". El sistema corre el query y envía el resultado por email o lo deja en la sección Guardados.

16

Grafo de relaciones entre documentos — de Obsidian

Visualización de cómo se relacionan los documentos de una colección por similitud semántica. Útil para explorar el conocimiento disponible y encontrar documentos relevantes antes de preguntar.

17

Extracción de action items — de Otter.ai / Fireflies

Después de una sesión larga, un botón "Extraer tareas" analiza la conversación y lista los puntos de acción mencionados. Se integra eventualmente con Jira, Notion Tasks, Linear.

18

Bot en Slack / Teams — de ChatGPT Enterprise

Instalás el bot en tu workspace de Slack o Teams. Los usuarios pueden hacer queries al RAG directamente desde el canal sin abrir la app. Las respuestas respetan el RBAC del sistema.

19

Autocompletado del input — de GitHub Copilot

Mientras el usuario tipea su pregunta, aparecen sugerencias de completado basadas en queries anteriores de la organización o del usuario. Tab para aceptar. Acelera la formulación de preguntas complejas.

20

Extracción estructurada a tabla — de Airtable + IA

El usuario define campos ("Nombre contrato, Fecha vencimiento, Monto, Parte") y el sistema extrae esa información de todos los documentos de una colección, generando una tabla exportable. Elimina trabajo manual masivo.
