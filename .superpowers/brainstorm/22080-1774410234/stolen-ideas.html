Ideas robadas de otros productos

Cada una está adaptada al contexto de un workspace RAG empresarial. Seleccioná todas las que te parecen interesantes — después las priorizamos.

1

Artifacts panel — de Claude

Cuando el RAG genera un documento, resumen o tabla, aparece en un panel lateral separado del chat. Se puede guardar, exportar, editar. El chat queda limpio; los artefactos viven aparte.

📄 Resumen generado
2

Proyectos con contexto persistente — de Claude

Agrupa sesiones dentro de un Proyecto. Un proyecto tiene colecciones asignadas, instrucciones custom ("siempre responder en formato ejecutivo") y archivos compartidos. Cada sesión hereda el contexto del proyecto.

3

Command palette — de Linear / Raycast

Cmd+K abre una paleta global. Desde ahí: nueva sesión, cambiar de colección, buscar en historial, navegar a admin, ejecutar acciones. Todo sin mouse. El producto se vuelve keyboard-first.

4

Panel de fuentes — de Perplexity

Cada respuesta tiene un panel lateral con los documentos fuente usados, con relevance score y fragment del texto que originó la respuesta. El usuario puede abrir el doc y ver exactamente qué párrafo se consultó.

5

Preguntas relacionadas — de Perplexity

Después de cada respuesta, aparecen 3-4 preguntas sugeridas relacionadas. Acelera la exploración y reduce la fatiga de formulación. Muy útil para usuarios no técnicos.

6

Templates de query — de Notion

El admin puede crear templates de prompts ("Resumen ejecutivo de contrato", "Análisis de riesgo regulatorio"). Los usuarios los usan como punto de partida. Consistency + velocity.

7

Respuestas ancladas / guardadas — de Slack

Marcar una respuesta como "guardada". Las respuestas guardadas tienen su propia vista. Útil cuando el RAG genera algo importante y no querés perderlo en el historial.

8

Compartir sesión — de ChatGPT

Genera un link read-only de una sesión para compartirla con alguien (dentro o fuera de la organización). El destinatario ve el diálogo pero no puede interactuar.

9

Modos de foco / modelo — de Perplexity

Antes de enviar un mensaje, el usuario elige el modo: "Detallado", "Ejecutivo", "Técnico", "Comparativo". Cada modo modifica el system prompt automáticamente.

10

Chat con un documento específico — idea propia

Desde el browser de colecciones, el usuario puede hacer click en un documento y abrir una sesión de chat anclada a ese documento. Equivalente a "Ask this PDF" pero dentro del sistema.
