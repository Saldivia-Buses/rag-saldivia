¿Cómo se comporta el panel Admin?

Cuando el admin hace click en el ícono Admin en la barra izquierda, ¿qué pasa con el panel secundario?

A

Panel secundario se convierte en nav admin

El panel de sesiones se reemplaza con el menú admin (Usuarios, Áreas, Permisos, Config RAG, Sistema). La jerarquía es clara: ícono izquierda elige la sección, panel central elige el sub-ítem. El contenido principal cambia según lo que selecciones en el panel.

B

Admin ocupa el espacio completo (sin panel secundario)

Al entrar a Admin, el panel secundario desaparece. La sección admin usa todo el ancho. El contenido tiene más espacio — mejor para tablas de usuarios, listas, formularios. La nav admin va dentro del propio contenido como tabs o breadcrumbs.
