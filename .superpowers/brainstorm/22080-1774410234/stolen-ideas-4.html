Ronda 4 — territorio experimental

Voz · Visualización · Privacidad · Branching · Personalización profunda

31

Entrada por voz — de Whisper / iOS dictado

Botón de micrófono en el input. Dictás tu pregunta, se transcribe en tiempo real, se envía. No requiere instalar nada — Web Speech API nativa en Chrome. Esencial para mobile y para usuarios que escriben poco.

32

Bifurcación de conversaciones — de git branches

En cualquier punto de la sesión, podés "bifurcar" y explorar una dirección distinta sin perder el camino anterior. Como hacer git branch desde un commit. Ideal para explorar hipótesis sin ensuciar el hilo principal.

Pregunta inicial
→
Rama A: análisis legal
Rama B: impacto financiero
33

Preview de documentos inline — de Google Drive viewer

Cuando el RAG cita un documento, hacés click y se abre un panel lateral con el PDF renderizado, con el fragmento citado resaltado en amarillo. Nunca abandonás el chat para verificar la fuente.

34

Visualizaciones generadas — de Code Interpreter / Julius AI

Cuando la respuesta contiene datos (fechas, montos, porcentajes, rankings), el sistema auto-renderiza un gráfico de barras, línea o tabla visual. El usuario puede cambiar el tipo de chart. Exportable como imagen.

35

Detección y redacción de PII — de AWS Comprehend / presidio

Antes de enviar un query al modelo, el sistema detecta información sensible (DNI, emails, teléfonos, nombres propios) y la reemplaza con tokens. El resultado se des-tokeniza antes de mostrar al usuario. Compliance sin fricción.

36

Memoria de usuario — de ChatGPT Memory

El sistema aprende y recuerda: tu rol, preferencias de formato, términos frecuentes, áreas de interés. "Recordar que prefiero respuestas en bullets" o "Siempre que menciones normativas, incluir la fecha". La UI muestra qué recuerda y te deja editarlo.

37

Detección de brechas de conocimiento — idea propia

Cuando el RAG no puede responder bien una pregunta, lo registra como "gap". El admin ve un dashboard de las preguntas sin respuesta agrupadas por tema — guía directa sobre qué documentos hay que ingestar para cubrir esas brechas.

38

White-labeling por organización — de HubSpot / Notion teams

Cada organización configura: logo propio, nombre del producto, paleta de colores, dominio personalizado (rag.miempresa.com). Los usuarios ven la herramienta como algo de la empresa, no como "RAG Saldivia". B2B game changer.

39

Modo comparación de documentos — de Diffchecker / contratos legales

Seleccionás dos documentos de una colección y abrís un modo comparación. El RAG responde preguntas sobre ambos en paralelo, mostrando diferencias y similitudes. Killer feature para análisis legal o de políticas.

40

PWA + borradores offline — de Notion / Bear mobile

La app se instala como PWA en el celular o laptop. Los queries se pueden escribir offline y se envían cuando hay conexión. Las sesiones pasadas están disponibles sin internet. El historial sobrevive sin servidor.

Llevamos 40 features exploradas. Cuando confirmes esta ronda, armamos el roadmap completo con todo lo que seleccionaste — organizado por fases, con estimación de esfuerzo y dependencias entre features.
