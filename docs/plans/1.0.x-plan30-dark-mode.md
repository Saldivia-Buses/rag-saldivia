## Qué se hizo

Fix de tokens dark mode, reemplazo de colores hardcodeados por tokens semánticos, y Mermaid theme-aware. Este es Plan 20 del roadmap original.

### Problema principal resuelto

`--accent-fg` en dark mode valía `#0c1929` (oscuro) pero el accent es `#60a5fa` (azul claro). 7+ componentes usaban `text-white` hardcodeado que funcionaba por accidente. Mermaid tenía 15+ hex hardcodeados mezclando tokens light y dark — bug visible.
