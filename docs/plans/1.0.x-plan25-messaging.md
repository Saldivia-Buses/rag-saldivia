## Qué se hizo

Sistema de mensajería interno completo: canales, DMs, threads, reacciones, mentions, pins, typing indicators, presencia, notificaciones, búsqueda FTS5, WebSocket sidecar.

### Stack

| Componente | Tecnología |
|---|---|
| Real-time | Bun native WebSocket sidecar en :3001 |
| Fan-out | Redis Pub/Sub |
| Presencia | Redis TTL keys (`presence:user:{id}` con 30s TTL) |
| Typing | Redis Pub/Sub (ephemeral, no DB) |
| Persistencia | SQLite (Drizzle) — `packages/db/src/schema/messaging.ts` |
| Búsqueda | SQLite FTS5 via raw SQL en init |
| Notificaciones | WebSocket + BullMQ |
| UI | React + shadcn/ui — `components/messaging/` |
