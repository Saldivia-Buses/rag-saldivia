## Frontend overhaul (plan 18 o sub-plan)

### Bloqueantes identificados

1. **Migrar a TanStack Query** — todas las 16 páginas usan useState/useEffect manual
2. **Error handling en mutations** — zero try/catch en operaciones de escritura
3. **Tabs faltantes** — 10+ tabs del plan spec sin implementar
4. **Create flows faltantes** — contable, ventas, facturación son read-only
5. **Search debounce** — API call on every keystroke
6. **Tildes españolas** — inconsistentes en toda la UI

### Approach propuesto

Hacer un plan 18 "Frontend Polish" que:
1. Extraiga helpers compartidos (fmtMoney, fmtDate, parseFloat)
2. Migre a TanStack Query (useQuery/useMutation por página)
3. Agregue error handling con toast (sonner)
4. Complete las tabs faltantes
5. Agregue create dialogs donde faltan
6. Normalice sugerencias al layout del resto
