# Plan 15 — UI Token Extraction + Branding Azure

> **Branch:** 1.0.x
> **Prerequisito:** Plan 13 (completado)
> **Sprint:** think → plan → execute → review → test → ship
> **Intensity:** standard

## Contexto

El design actual "Warm Intelligence" tiene fondos crema (#faf8f4), bordes
amarillentos (#ede9e0) y acento navy (#1a5276). Se ve genérico.

Enzo quiere: tokens inspirados en Claude.ai, acento azure blue, branding
"Saldivia RAG". NO clonar literal — extraer la sensación de limpieza y
neutralidad de Claude, aplicar con identidad propia.

## Enfoque

Tres cambios principales en `globals.css`:
1. **Backgrounds:** menos crema, más neutral (ivory limpio)
2. **Accent:** navy → azure blue (#2563eb rango)
3. **Branding:** "Saldivia RAG" consistente en login y NavRail

No se tocan componentes — solo tokens CSS + texto de branding.

## Scope

**Archivos a modificar:**
- `apps/web/src/app/globals.css` — tokens light + dark
