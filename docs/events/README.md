# Event Catalog

Auto-generated documentation for all typed spine events. Source of truth is
`pkg/events/spec/*.cue`; these docs are regenerated by `make events-gen`.

## Families

| Family | File | Events |
|--------|------|--------|
| [notify](./notify.md) | `pkg/events/spec/notify.cue` | `chat.new_message`, `auth.login_success` |
| [ingest](./ingest.md) | `pkg/events/spec/ingest.cue` | `ingest.completed` |
| [platform](./platform.md) | `pkg/events/spec/platform.cue` | `platform.lifecycle` |

## Adding events

1. Edit or create a `.cue` file in `pkg/events/spec/`.
2. Run `make events-gen`.
3. Commit spec + generated artifacts.
4. See `docs/conventions/cue.md` for breaking change rules.

## Regenerating

```bash
make events-gen       # regenerate Go/TS/docs
make events-validate  # CI guard — fails if out of sync
```
