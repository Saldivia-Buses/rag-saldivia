## Reproduction

```bash
# 1. Histrix tables (skip header row)
tail -n +2 .intranet-scrape/db-tables.txt | sort -u > /tmp/histrix_tables.txt

# 2. Tables covered by any reader / migrator (broad grep — catches both
#    named-struct and positional-literal declarations).
grep -rhoE '"[A-Z][A-Z0-9_]{3,}"' \
  tools/cli/internal/legacy/*.go tools/cli/internal/migration/*.go \
  | sed 's/"//g' | sort -u > /tmp/all_lit.txt
comm -12 /tmp/all_lit.txt /tmp/histrix_tables.txt > /tmp/legacy_covered.txt

# 3. Row counts from live Histrix (requires WireGuard VPN up +
#    172.22.100.99 SSH reachable). Write query.sql and run.sh once:
cat > /tmp/q.sql <<'SQL'
SELECT table_name, table_rows FROM information_schema.tables
WHERE table_schema='saldivia' ORDER BY table_rows DESC;
SQL
docker run --rm --network=host -v /tmp:/work alpine:3.20 sh -c '
  apk add --no-cache openssh-client mariadb-client sshpass >/dev/null 2>&1
  sshpass -p "SaldiviaAdmin02" ssh \
    -o StrictHostKeyChecking=no -o UserKnownHostsFile=/dev/null \
    -o LogLevel=ERROR sistemas@172.22.100.99 \
    "mysql -u root -pm2450e --batch --skip-column-names information_schema" \
    < /work/q.sql
' | grep -E "^[A-Z][A-Z0-9_]*	[0-9]+" > /tmp/rowcounts.tsv

# 4. Diff + join
comm -23 /tmp/histrix_tables.txt /tmp/legacy_covered.txt > /tmp/uncovered.txt
awk -F'\t' 'NR==FNR{r[$1]=$2; next} {print (r[$1]+0) "\t" $1}' \
  /tmp/rowcounts.tsv /tmp/uncovered.txt \
  | sort -k1 -n -r > /tmp/uncovered_ranked.tsv
```

The top of `/tmp/uncovered_ranked.tsv` is the living priority list. The
zero-row entries at the bottom are the candidates for the next bulk
waiver revision.
